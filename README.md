# Quicktoast
Quickest way to show a toast in android!

# Installation
<b>Add it in your root build.gradle :</b>
``` allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
<b>Add the dependency:</b>

```
dependencies {
	        implementation 'com.github.meet30997:Quicktoast:1.0.0'
	}
  ```
